# IMDb Movie Dataset Project
This project focused on building an etl pipeline for the IMDb Dataset
The following tools were used
1. Amazon S3 Buckets
2. Amazon EMR Clusters
3. Amazon Redshift
4. Amazon Quicksight
5. Microsoft PowerBi
